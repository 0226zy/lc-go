# 2. 两数相加 (Add Two Numbers)

## 题目描述

给你两个非空的链表，表示两个非负的整数。它们每位数字都是按照逆序的方式存储的，并且每个节点只能存储一位数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

### 示例 1

```
输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807
```

### 示例 2

```
输入：l1 = [0], l2 = [0]
输出：[0]
```

### 示例 3

```
输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]
```

## 提示

- 每个链表中的节点数在范围 `[1, 100]` 内
- `0 <= Node.val <= 9`
- 题目数据保证列表表示的数字不含前导零

## 题目解析

### 核心思路

这道题是典型的大数加法模拟问题，核心在于模仿小学学习的 **竖式加法过程**：

1. **逆序存储的优势**：由于数字是逆序存储（个位在前），可以直接从头节点开始逐位相加，无需翻转链表

2. **进位处理**：使用 `carry` 变量记录进位（0 或 1），因为两个一位数相加最大为 `9+9+1=19`，所以进位最多为 1

3. **链表长度不同**：当其中一个链表遍历完后，将其值视为 0 继续计算

4. **最终进位**：遍历结束后如果 `carry > 0`，需要额外添加一个节点

### 算法步骤

1. 初始化虚拟头节点 `dummy` 和当前节点指针 `cur`
2. 初始化进位 `carry = 0`
3. 同时遍历两个链表，直到两者都为空：
   - 获取当前位的值（链表为空则取 0）
   - 计算和：`sum = n1 + n2 + carry`
   - 创建新节点：`digit = sum % 10`
   - 更新进位：`carry = sum / 10`
4. 遍历结束后，若 `carry > 0`，追加新节点
5. 返回 `dummy.next`

### 复杂度分析

- **时间复杂度**: O(max(m, n))，其中 m 和 n 分别为两个链表的长度
- **空间复杂度**: O(max(m, n))，返回值不计入空间复杂度则为 O(1)
